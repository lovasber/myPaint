# MY DRAWING APP
Hello I made a simple drawing app in React (typescript) and with the help of roughJS ‚ù§.

## Features
 - Drawing straight line
 - Drawing rectangle
 - Moving shape
 - Select to delete shape
	 - First select the shape and press the `delete` key on your keyboard

## RUN

`npm install`
`npm run dev`

## BUILD 
`npm run-script build`
